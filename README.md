## Requirements

- Python >= 3.7


## Install Locally Or On A VPS

```sh
$ git clone https://github.com/PunyaChael/SecurityGroupBot

$ cd SecurityGroupBot

$ pip3 install -U -r requirements.txt

$ cp sample_config.py config.py
```
Edit **config.py** with your own values

# Run Directly
```sh
$ python3 -m PunyaChael
```
